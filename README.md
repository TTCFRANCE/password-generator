# Générateur de mot de passe (Python - Console)

## Description
Ce projet est un générateur de mots de passe aléatoires en ligne de commande.  
Il permet de créer des mots de passe sécurisés en combinant lettres, chiffres et caractères spéciaux.

## Objectif
J'ai réalisé ce projet pour apprendre :
- l'utilisation de modules Python (`random`, `string`)
- la génération aléatoire
- la gestion des entrées utilisateur

## Fonctionnalités
- Génération de mots de passe aléatoires
- Choix de la longueur du mot de passe
- Utilisation de différents types de caractères :
  - lettres majuscules
  - lettres minuscules
  - chiffres
  - caractères spéciaux

## Fonctionnement
1. L'utilisateur choisit la longueur du mot de passe
2. Le programme génère un mot de passe aléatoire
3. Le résultat est affiché

## Technologies utilisées
- Python

## Lancer le projet
1. Installer Python
2. Exécuter le fichier :
```bash
python generateur_mot_de_passe_console.py
